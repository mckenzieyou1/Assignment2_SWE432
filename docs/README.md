# Assignment2_SWE432
## Overview

Static site with the core screens per role (Producer, DJ, Manager). Includes:

- **HTML5** with lists, tables, and forms with native validation.

- **CSS** external + internal + inline, using Grid (layout) and Flexbox (toolbars).

- **Basic accessibility:** label/for, fieldset/legend, caption, aria-live.

## How to run

1. Clone or download the repo.

2. Open index.html in your browser.

3. Navigate to each role from the top bar/aside.

## Pages

- /index.html (home + navigation)

- /producer.html (playlist planning)

- /dj.html (on-air queue, quick add, notes)

- /manager.html (assignments, add DJ, metrics)
